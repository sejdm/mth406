  2. For each language that you proved was regular in the previous exercise sets, find a context free grammar to define it. Can any regular language be defined by context free grammar?

    1. \(\{0^n1^n \ |\ n=0,1,\ldots\}\)

    2. The complement of the previous language, i.e. \(\{0^m1^n \ |\ m>n,m,n=0,1,\ldots\}\)

  4. What language does the context free grammar \(G:=(\{X\},\{(,)\},R,X)\), where \(R=\{X\to (X) | XX | Ïµ\}\) generate?
